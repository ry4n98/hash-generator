## 1. Install hash-gen

```bash
npm install -g hash-gen
```
## 2. Start Hashing!


### App options

```
    -h, --hash      Type of hash e.g sha256, sha512 etc. (sha256 is default)
    -t, --target    The hash you are trying to find a match for.
    -s, --start     The number you would like to start at (1 is default).
```
